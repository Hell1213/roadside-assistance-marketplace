# Gilroy Font Files

Please add the following Gilroy font files to this directory:

- Gilroy-Regular.ttf (weight: 400)
- Gilroy-Medium.ttf (weight: 500)
- Gilroy-SemiBold.ttf (weight: 600)
- Gilroy-Bold.ttf (weight: 700)

You can download Gilroy font from:
- Google Fonts (if available)
- Font repositories like DaFont
- Purchase from official font foundries

For now, the apps will fall back to system fonts until these files are added.
